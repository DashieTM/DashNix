# List of keymaps

## Space as leader

## Debugging
- F5: toggle breakpoint
- F6: step over breakpoint
- F7: step into breakpoint
- F8: start debugging
- F9: continue from breakpoint
- F10: close debug UI

## Nerd Tree
- t: toggle open
- f: focus tree

## buffer switching
- F1: next buffer (cycles)
- F2: previous buffer (cycles)

## formatting
- F4: format this file

## telescope
- ff: find files
- fg: live ripgrep
- fb: find buffers
- fh: help for functions etc

## toggletrouble
- Ctrl + f: show errors and warnings
